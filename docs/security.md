# Security Checklist (минимальный)

- Логи: JSON-формат, PII-маскирование, нет секретов
- Конфиг: секреты не логгируются, переменные ENV валидируются
- Трассировка: `trace_id` в логах для корреляции
- Дедупликация: предотвращает повторную обработку
- Retriес: ограничены, без бесконечных попыток
- DLQ: причина и ссылка на исходный payload присутствуют
